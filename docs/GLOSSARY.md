# 📖 Глоссарий

## AI Team System

### Агенты
- **TeamLead** — глава команды, координирует работу
- **Architect** — проектирует архитектуру
- **Backend** — пишет серверный код
- **Frontend** — создаёт интерфейс
- **DevOps** — настраивает инфраструктуру
- **Tester** — пишет тесты
- **Documentalist** — пишет документацию

### Профили железа
- **Light** — слабое железо (2 агента)
- **Medium** — среднее железо (4 агента)
- **Heavy** — мощное железо (8 агентов)

### Термины
- **Context window** — максимальный размер входных данных для модели
- **Temperature** — параметр "креативности" модели
- **Ollama** — локальный сервер для запуска LLM
- **Prompts** — инструкции для AI агентов

### Файлы
- `requirements.md` — требования к проекту
- `system_prompt.md` — главный системный промпт
- `profiles.yaml` — конфигурация профилей
- `ai_team.db` — SQLite база данных
