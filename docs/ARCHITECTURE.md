## Компоненты

### Core модули

| Модуль | Назначение |
|--------|------------|
| `main.py` | Главный оркестратор, управляет потоком |
| `agent_manager.py` | Запуск и управление агентами |
| `model_router.py` | Маршрутизация к Ollama/API |
| `database.py` | SQLite база данных |
| `logger.py` | Логирование событий |
| `system_scanner.py` | Определение железа |
| `git_manager.py` | Git операции |

### Агенты

| Агент | Роль |
|-------|------|
| TeamLead | Координация, планирование |
| Architect | Архитектура, технологии |
| Backend | Серверный код |
| Frontend | Интерфейс |
| DevOps | Docker, CI/CD |
| Tester | Тесты |
| Documentalist | Документация |

## Поток данных

1. **Input**: requirements.md
2. **TeamLead**: Анализ → План задач
3. **Architect**: Архитектура → Структура проекта
4. **Workers**: Параллельное создание кода
5. **Output**: Готовый проект + отчёт

## Профили железа

```yaml
light:   # 2 агента, 4K ctx, Ollama only
medium:  # 4 агента, 8K ctx, Ollama + API fallback
heavy:   # 8 агентов, 16K ctx, API primary
```
