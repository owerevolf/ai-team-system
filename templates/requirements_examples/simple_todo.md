# Простой Todo List

## Описание
Веб-приложение для управления задачами (todo list).

## Требования
- Добавление задач
- Отметка выполненных
- Удаление задач
- Сохранение в localStorage
- Фильтрация (все/активные/завершённые)

## Стек
- HTML/CSS/JavaScript
- Без фреймворков

## Функционал
1. Поле ввода + кнопка "Добавить"
2. Список задач с чекбоксами
3. Кнопка удаления
4. Счётчик активных задач
5. Кнопки фильтрации

## Дизайн
- Минималистичный
- Адаптивный
- Тёмная тема
