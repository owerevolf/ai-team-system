# Сайт-портфолио

## Описание
Персональный сайт-портфолио разработчика.

## Требования
- HTML + CSS + JavaScript
- Адаптивный дизайн
- Тёмная и светлая тема
- Анимации при скролле

## Секции
1. Hero — имя, фото, краткое описание
2. About — о себе
3. Skills — навыки (с прогресс-барами)
4. Projects — карточки проектов
5. Contact — форма обратной связи

## Стили
- CSS Variables для тем
- Flexbox + Grid
- Smooth scroll
- Media queries для мобильных

## Файлы
- index.html
- style.css
- app.js
