### ШАГ 3: Функциональное тестирование
```
Тест 1: Базовая работа
1. Нажать кнопку "2"

4. Нажать кнопку "="
5. Проверить что в поле ввода появилось "5"
6. Скриншот 2: "Результат 2+3=5"

Тест 2: Обработка ошибок
1. Нажать кнопку "=" без ввода
2. Проверить что НЕ упало
3. Проверить что есть сообщение об ошибке или пустой результат
4. Скриншот 3: "Обработка пустого ввода"

Тест 3: Интерфейс
1. Проверить что кнопки кликабельны
2. Проверить что текст читаемый
3. Проверить что нет наложений элементов
4. Скриншот 4: "Интерфейс крупным планом"
```

### ШАГ 4: Проверка консоли
```
Проверить консоль браузера:
- Нет ошибок (error)
- Нет предупреждений (warning) — желательно
- Нет 404 для ресурсов
```

## ИНСТРУМЕНТЫ (Playwright API)
## ═══════════════════════════════════════

```python
# Запуск браузера
browser = await playwright.chromium.launch()
page = await browser.new_page()

# Открыть страницу
await page.goto('http://localhost:5000')
await page.wait_for_timeout(3000)

# Скриншот
await page.screenshot(path='screenshot_1.png')

# Нажать кнопку
await page.click('text="2"')

# Ввести текст
await page.fill('input#display', '42')

# Получить текст
text = await page.text_content('h1')

# Проверить консоль
page.on('console', lambda msg: print(f'Console: {msg.text}'))

# Проверить ошибки
page.on('pageerror', lambda err: print(f'Error: {err}'))
```
