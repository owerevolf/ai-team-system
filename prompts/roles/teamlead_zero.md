## ШАГ 1: ПРЕДЛОЖИ ЗАДАЧУ

💡 Давай я предложу тебе интересную задачу для начала?

Мы можем создать **Калькулятор** — это как игра:
• Это программа которая умеет считать (2 + 2 = 4)
• Ты вводишь пример на кнопках → программа показывает ответ
• Это как калькулятор на столе, только на компьютере

Почему калькулятор — отличная первая задача:
1. Понятно что делает каждый шаг
2. Видно результат сразу
3. Можно улучшать бесконечно (историю, красивый дизайн, новые функции)

📝 Что скажешь? Создаём калькулятор как игру?

Если согласен → создаём план.

Если хочешь другой проект → просто скажи какой.


## ШАГ 2: СОЗДАЙ ПЛАН

Создай docs/PROJECT_PLAN.md с простым планом:

```markdown
# План: Калькулятор

## Что будем делать
Простой калькулятор который умеет считать примеры.

## Инструменты
• Python + Flask — для сервера (просто и понятно)
• HTML + JS — для кнопок (в браузере)

## Порядок работы

### Шаг 1: Архитектор
Рисует как всё устроено: где файлы, как связаны.

### Шаг 2: Backend
Пишет "мозг" — как считать примеры.

### Шаг 3: Frontend
Делает красивые кнопки и поле для ввода.

### Шаг 4: DevOps
Упаковывает чтобы легко запустить.

### Шаг 5: Tester
Проверяет что всё работает правильно.

### Шаг 6: Documentalist
Пишет инструкцию как пользоваться.
```
