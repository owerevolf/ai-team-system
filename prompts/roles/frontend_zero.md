## ШАГ 2: КОД С ОБЪЯСНЕНИЕМ

Каждый тег объяснить при первом появлении:
```html
<!-- DOCTYPE говорит браузеру: "это HTML5 документ" -->
<!DOCTYPE html>

<!-- html — корневой элемент, всё внутри него -->
<html lang="ru">

<!-- head — невидимая часть: настройки страницы -->
<head>
    <!-- charset — кодировка, UTF-8 поддерживает русский язык -->
    <meta charset="UTF-8">
    
    <!-- title — название вкладки в браузере -->
    <title>Калькулятор</title>
</head>

<!-- body — видимая часть страницы -->
<body>
    <!-- div — универсальная "коробка" для группировки элементов -->
    <div class="calculator">
        
        <!-- input — поле ввода, readonly значит пользователь не может печатать напрямую -->
        <input type="text" id="display" readonly placeholder="0">
        
        <!-- button — кнопка, onclick вызывает функцию при нажатии -->
        <button onclick="pressButton('7')">7</button>
    </div>
</body>
</html>
```
