## ШАГ 2: КОД С ОБЪЯСНЕНИЕМ КАЖДОЙ СТРОКИ

Пример как писать код:
```python
# Импортируем Flask — наш веб-фреймворк (набор инструментов для веба)
from flask import Flask, request, jsonify

# Создаём приложение Flask — это и есть наш "сервер"
app = Flask(__name__)

# @app.route — говорим Flask: "когда придёт запрос на /calculate, вызови эту функцию"
# methods=['POST'] — принимаем только POST запросы (когда данные отправляют)
@app.route('/calculate', methods=['POST'])
def calculate():
    # Получаем данные от пользователя (выражение которое нужно посчитать)
    data = request.get_json()
    
    # Берём поле 'expression' из полученных данных
    expression = data.get('expression', '')
    
    try:
        # eval() — Python вычисляет математическое выражение
        # Осторожно: в реальных проектах eval() опасен, используем safe_eval
        result = eval(expression)
        
        # jsonify превращает Python-словарь в JSON (формат данных для интернета)
        return jsonify({'result': result, 'expression': expression})
    
    except ZeroDivisionError:
        # Если пользователь делит на ноль — возвращаем понятную ошибку
        return jsonify({'error': 'На ноль делить нельзя!'}), 400
    
    except Exception:
        # Любая другая ошибка — говорим что выражение неверное
        return jsonify({'error': 'Неверное выражение'}), 400
```
