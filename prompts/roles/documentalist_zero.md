# Калькулятор

Простой калькулятор с историей и конвертером валют.
Работает в браузере, запускается одной командой.

## Что умеет

- Считать: +, -, *, /, скобки, проценты
- Помнит историю (последние 50 вычислений)
- Конвертирует валюты (доллары ↔ рубли)
- Красивый интерфейс, тёмная тема

## Быстрый старт

### Способ 1: Без Docker (проще)

**Шаг 1.** Открой терминал (на Windows — PowerShell)

**Шаг 2.** Перейди в папку проекта:
```bash
cd calculator
```

**Шаг 3.** Поставь библиотеки:
```bash
pip install -r requirements.txt
```
Увидишь текст "Successfully installed..." — это значит поставилось.

**Шаг 4.** Запусти:
```bash
python src/main.py
```
Увидишь: "Running on http://127.0.0.1:5000"

**Шаг 5.** Открой браузер и перейди по адресу:
```
http://localhost:5000
```
Увидишь калькулятор. Готово!

### Способ 2: С Docker (если установлен)

```bash
docker build -t calculator .
docker run -p 5000:5000 calculator
```
Потом так же открой http://localhost:5000

## Структура проекта

```
calculator/
├── src/
│   ├── main.py         # Сервер, запуск
│   ├── calculator.py   # Сложение, вычитание и т.д.
│   ├── history.py      # Хранение истории
│   └── converter.py    # Конвертер валют
├── templates/
│   └── index.html      # Страница калькулятора
├── static/
│   ├── style.css       # Красота (цвета, шрифты)
│   └── script.js       # Кнопки работают
├── tests/              # Тесты (проверка что работает)
├── Dockerfile          # Рецепт для Docker
└── requirements.txt    # Список библиотек
```

## Если что-то не работает

**"pip не найден"** — поставь Python с python.org, выбери "Add to PATH"

**"Module not found"** — пропусти Шаг 3, попробуй снова

**"Port already in use"** — что-то уже на порту 5000. Закрой или поменяй порт в main.py
```
