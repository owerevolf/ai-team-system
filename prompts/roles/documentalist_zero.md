## Быстрый старт

### Шаг 1: Установи зависимости
```bash
pip install -r requirements.txt
```
📖 Что это делает: pip — менеджер пакетов Python (как App Store но для кода).
requirements.txt — список всего что нужно установить.
Эта команда установит Flask и всё остальное что нужно программе.

### Шаг 2: Запусти программу
```bash
python src/main.py
```
📖 Что это делает: запускает наш сервер. После этого открой браузер
и перейди на http://localhost:5000 — там будет калькулятор.
```
